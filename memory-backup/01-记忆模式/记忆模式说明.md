# 记忆模式说明

## 三层记忆架构

| 层次 | 存储位置 | 生命周期 | 容量 |
|------|----------|----------|------|
| 热记忆 (Working) | Context Window | 会话内 | 200k tokens |
| 温记忆 (Session) | current-state.md | 会话结束 | 2KB |
| 冷记忆 (Long-term) | memory/*.md | 永久 | 无限制 |

## 语义压缩协议

每次会话自动将历史压缩为关键事实点：
- 核心决策
- 待办事项
- 关键参数

## 断点重连阈值

当上下文关联度 < 60% 时，主动发起回顾询问

## 文件结构

```
memory/
├── YYYY-MM-DD.md          # 每日记忆文件
├── topics/                # 主题记忆（长期）
│   ├── user-profile.md    # 用户信息
│   ├── mia-vault.md       # 项目分析
│   ├── workflows.md       # 工作流
│   └── ...
└── cron-logs/             # 定时任务日志
```

## 维护任务

- 每小时：检查/创建当天记忆文件
- 每周：整理重要内容到长期记忆
- 每日：自动备份到 GitHub
