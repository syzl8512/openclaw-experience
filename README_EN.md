# Claude Code Custom Commands

This repository contains custom commands for Claude Code to improve work efficiency and automate workflows.

---

## Directory Structure

```
openclaw-experience/
├── claude-commands/
│   └── ppt制作.md       # AI PPT Generator
└── README.md
└── README_EN.md
```

---

## Quick Start

### Installation

1. Clone this repository:
```bash
git clone https://github.com/syzl8512/openclaw-experience.git
```

2. Copy command file to Claude Code commands directory:
```bash
# macOS
cp claude-commands/ppt制作.md ~/.claude/commands/
```

---

## Commands

### /ppt制作 - AI PPT Generator

**Function:** Generate 乔布斯-style presentations from manuscripts or adjust existing PPTs.

**Usage:**

```
/ppt制作
[Your manuscript content]
[Target audience: xxx]
```
