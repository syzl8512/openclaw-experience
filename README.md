# OpenClaw 经验技能库

> 通过与 Mia 的对话学习积累的技能和经验

## 目录结构

```
openclaw-experience/
├── reback/              # 回滚保护系统
├── file-sender/        # 飞书文件发送
├── brainstorm/         # 头脑风暴技能
├── writing-plans/      # 任务规划技能
├── workflow-runner/    # 工作流执行技能
├── doc-writer/        # 文档生成技能
├── skill-tools/        # 技能工具
│   ├── find-skills-usage.md    # Find-Skills 使用指南
│   └── skill-creator-usage.md  # Skill-Creator 使用指南
└── mineru-cloud/       # MinerU PDF 解析云端版
```

## 内容说明

### reback
回滚保护系统，用于远程修改配置时的自动回滚保护。

### file-sender
飞书文件发送技能，包含如何发送文件到飞书的各种方法和注意事项。

### brainstorm
头脑风暴技能，参考 Superpowers 创建。用于在实施任何任务前充分理解需求。

### writing-plans
任务规划技能，将需求拆分成可执行的小任务。

### workflow-runner
工作流执行技能，支持串行/并行执行任务。

### doc-writer
文档生成技能，模板化生成各类文档。

### skill-tools
技能工具相关：
- find-skills-usage.md: 查找和安装技能的使用指南
- skill-creator-usage.md: 创建自己技能的使用指南

### mineru-cloud
MinerU 云端 API 技能，用于 PDF 文档智能解析。

## 经验分享机制

当你说"经验分享"时，会自动：
1. 收集新增的技能和经验
2. 更新到本地文件夹
3. 推送到 GitHub
4. 检查并下载新技能
