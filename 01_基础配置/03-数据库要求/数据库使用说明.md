# 数据库相关要求

## 当前使用的数据库

### 1. LanceDB (向量数据库)

**用途**：向量存储、语义搜索

**安装**：
```bash
pip install lancedb
```

**使用示例**：
```python
import lancedb
import pyarrow as pa

# 连接数据库
db = lancedb.connect("/tmp/lancedb_test")

# 定义 schema
schema = pa.schema([
    pa.field("id", pa.int64()),
    pa.field("name", pa.string()),
    pa.field("score", pa.int64()),
])

# 创建表
table = db.create_table("test_table", schema=schema, mode="overwrite")

# 写入数据
table.add([{"id": 1, "name": "Alice", "score": 95}])

# 读取数据
results = table.to_pandas()
```

**特点**：
- 本地文件存储，无需额外服务
- 支持向量索引
- Python 原生支持

### 2. Supabase (云数据库)

**用途**：云端存储、用户数据、pgvector 向量搜索

**配置**：
- 需要 Supabase 项目 URL 和 API Key
- 使用 `supabase` skill 进行操作

### 3. SQLite (本地数据库)

**用途**：轻量级本地存储

**位置**：`/Users/agent/.openclaw/workspace/*.db`

## 数据备份

- 记忆文件：每日自动备份到 GitHub
- LanceDB：存储在 `/tmp/lancedb_test/`
- 配置文件：`~/.openclaw/config.yaml`

## 注意事项

1. LanceDB 需要 PyArrow 依赖
2. 向量字段需要指定维度
3. 定期清理临时数据
